# Shrinivas Rental App

A fully functional property rental web application built using **HTML, CSS, JavaScript, PHP, and MySQL**. 

## 🏠 Features
- **Browse Properties**: Users can view available houses and apartments for rent.
- **Add Property**: Easy submission of new properties to the platform.
- **Admin Dashboard**: Backend admin panel for managing properties, users, and site settings.
- **User Authentication**: Secure user registration and login.
- **Responsive Design**: A modern, clean, and functional user interface.

## 🚀 Setup Instructions (Local Environment)
1. Install [XAMPP](https://www.apachefriends.org/) (or any similar local server stack).
2. Clone this repository into your `htdocs` directory:
   ```bash
   git clone https://github.com/mitdeveloper10/shrinivas-rental-app.git
   ```
3. Start the **Apache** and **MySQL** modules from the XAMPP Control Panel.
4. Go to `http://localhost/phpmyadmin` and create a new database named `rental_db`.
5. Import the provided `rental_db.sql` file into your newly created database.
6. Open your browser and navigate to `http://localhost/shrinivas` to view the application!

## 💻 Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: PHP
- **Database**: MySQL
